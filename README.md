# katlong-site

Personal portfolio site — plain HTML, CSS, and vanilla JS. No frameworks, no build step.

## Structure

```
public/   ← all site files served by GitHub Pages
```

## Sections

`#hero` · `#about` · `#skills` · `#projects` · `#repos` · `#contact`

## Deploy

Push to `main`. GitHub Actions publishes `public/` to GitHub Pages automatically.
